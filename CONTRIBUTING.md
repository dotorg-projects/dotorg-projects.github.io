# Contributing

Thank you for your interest in contributing to the .ORG Projects Website!

## Reporting Issues

If you find a bug or have a suggestion, please [open an issue](https://github.com/dotorg-projects/dotorg-projects.github.io/issues) on GitHub.

## Pull Requests

1. Fork the repository
2. Create a branch from `main`
3. Make your changes
4. Ensure all tests pass
5. Submit a pull request
